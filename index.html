Vue Swatches

Help the user picking beautiful colors

Get Started

Built-in Presets

Includes few collections of swatches for common scenarios like text editing.

Built-in Popover

Includes a builit-in popover with always-on-screen functionality.

Inline Mode

You can get rid of the built-in popover and show the swatches in your custom UI.

Easily Customizable

You can customize every aspect of the component using props and slots.

SSR Compatibility

You can use it at server side or in a static generated page. Easy to integrate with Nuxt.js.

Accessibility Support

Support for alternative text, currently fully accessible with the inline mode
